¿Qué es el cableado horizontal?

El cableado horizontal es la columna vertebral que conecta todos los dispositivos y servicios en una empresa. Se trata de un conjunto de cables y dispositivos que se encargan de transportar la información de un lugar a otro dentro de un edificio o recinto. Su función principal es interconectar las salidas de los paneles de parcheo, también conocidos como patch panels, de los cuartos de telecomunicaciones con las tomas de usuario en el área de trabajo.

¿Qué es el cableado vertical o backbone?

El cableado backbone, cableado vertical o troncal es una parte del cableado estructurado de un edificio, sirviendo de conexión entre el rack de telecomunicaciones principal (MDF o Main Distribution Frame) y el rack de telecomunicaciones secundario (HCn u Horizontal Cable) que se distribuyen por el edificio.

Tipos de cables para el sistema cableado horizontal

Cables de par trenzado sin blindar

es un tipo de cable de red compuesto por pares de hilos de cobre trenzados sin protección adicional. Es económico, flexible y comúnmente usado en redes LAN Ethernet para conexiones de datos y voz.

Cable TFP

es un tipo de cable de par trenzado que incluye una pantalla de aluminio para reducir la interferencia electromagnética

Cable STP

es un tipo de cable de red que incluye una pantalla metálica alrededor de cada par de hilos trenzados

Cables de fibra óptica multimodo

Es un tipo de fibra óptica utilizada principalmente para comunicaciones a corta distancia, como dentro de edificios o en campus universitarios .

El cable de fibra óptica monomodo

El cable de fibra óptica monomodo es utilizado en la industria de las telecomunicaciones debido a sus características. Transporta un único modo de luz, lo que permite la transmisión de datos a larga distancia y con un gran ancho de banda

Diferencias Principales

La principal diferencia entre T568A y T568B es la disposición de los pares de cables naranja y verde. Ambos estándares son funcionalmente equivalentes, pero es importante elegir uno y mantener la consistencia en toda la instalación para evitar problemas de compatibilidad.

Aplicación en Cables

  • Cable Directo: Ambos extremos del cable deben seguir el mismo estándar (T568A a T568A o T568B a T568B).
  • Cable Cruzado: Un extremo del cable sigue el estándar T568A y el otro extremo sigue el estándar T568B.

Opciones Adicionales

  • Compatibilidad: T568B es compatible con versiones anteriores, mientras que T568A no lo es.
  • Uso en Redes: T568A se utiliza más en redes de telefonía antiguas, mientras que T568B es preferido para nuevas instalaciones de red.

9 Para que se utiliza el Cable en la Red

Conexión de Dispositivos

  • Ordenadores: Conectar ordenadores a routers, switches o hubs para acceder a internet o a una red local.
  • Impresoras: Conectar impresoras de red para que puedan ser utilizadas por múltiples dispositivos.
  • Servidores: Conectar servidores a la red para compartir recursos y servicios.

Transmisión de Datos

  • Alta Velocidad: Los cables de red permiten la transmisión de datos a alta velocidad, esencial para aplicaciones que requieren gran ancho de banda.
  • Estabilidad: Ofrecen una conexión más estable y confiable en comparación con las conexiones inalámbricas.

Infraestructura de Red

  • Interconexión de Equipos de Red: Conectar routers, switches y otros equipos de red entre sí para crear una red más amplia y compleja.
  • Expansión de Redes: Permiten la expansión de la red a diferentes áreas de un edificio o campus.

Tipos de Cables de Red

  • UTP: El tipo más común, utilizado en la mayoría de las redes domésticas y comerciales.
  • STP: Similar al UTP pero con una capa adicional de protección contra interferencias electromagnéticas.
  • Fibra Óptica: Utilizado para conexiones de larga distancia y alta velocidad.

11 Para que se usa el Cableado Cruzado en la Red

Conexión Directa entre Dispositivos

  • Ordenador a Ordenador: Permite la transferencia de datos directamente entre dos ordenadores.
  • Switch a Switch: Conecta dos switches para expandir la red.
  • Router a Router: Conecta dos routers en una red.

Configuración de Redes Locales

  • Redes Pequeñas: Ideal para configurar redes pequeñas y temporales, como en entornos de prueba o para compartir archivos entre dos dispositivos sin necesidad de un router.

Configuración del Cableado Cruzado

Para crear un cable cruzado, un extremo del cable debe seguir el estándar T568A y el otro extremo debe seguir el estándar T568B.
